# How does it work 

1. git clone ...
2. cd clean_folder
3. python3 -m build
4. pip3 list | grep clean_folder  (check install module)
5. pip3 install -e .
6. Usage:
```
  clean_folder <PATH_to_DIR_which_need_to_sort>
```
